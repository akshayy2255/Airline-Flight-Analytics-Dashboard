✈ Airline Flight Analytics Dashboard
📊 Project Overview

This project presents an interactive dashboard built using Power BI to analyze airline flight data, uncovering insights on revenue, travel trends, and airline performance.

Objective

The goal of this project is to:
- Analyze airline performance  
- Identify high-revenue routes and patterns  
- Understand seasonal travel trends  
- Provide actionable business insights  

🛠 Tools & Technologies

- Power BI  
- DAX (Data Analysis Expressions)  
- Excel / CSV  

📂 Dataset Description

The dataset includes:
Airline names
Source and destination cities
Ticket price
Number of stops
Duration of flights
Journey dates

📈 Dashboard Features

KPI Cards: Total Revenue (97M), Average Price (9.09K), Total Duration (7M)
Revenue analysis by stop count
Flights distribution by source and destination
Monthly travel trends
Airline performance comparison
Interactive filters (Morning, Afternoon, Night)

🔍 Key Insights

- Delhi is the busiest origin city with the highest number of flights (~4K)  
- Jet Airways leads in both flight count and revenue  
- Single-stop flights generate the highest revenue  
- Travel demand peaks in May and June (32% each)  
- Cochin is the most frequent destination (~42%)
